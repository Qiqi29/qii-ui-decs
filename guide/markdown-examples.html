Markdown扩展示例 ​

本页展示了VitePress提供的一些内置扩展。

语法突出显示 ​

VitePress提供语法高亮显示，由 Shiki 提供, 具有线条突出显示等附加功能：

Input

md
```js{4}
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
```

Output

js
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
